3. 챔피언 등록하기
라인, 아이템, 룬, 스펠, 공백없이 정확하게 기입해야한다.
특히 아이템은 띄어쓰기 없이 기입해야한다.
라인 ex) 탑, 정글, 미드, 원딜, 서포터
아이템 ex) 돌풍, 불멸의철갑궁
룬 ex) 지배, 영감, 정밀, 마법, 결의
스펠 ex) 텔레포트, 점화, 점멸, 방어막, 정화
